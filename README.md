# Entropy
> **Restore the human chaos.**

**Entropy** is an advanced AI humanizer designed to break the sterile, predictable patterns of machine-generated text. By injecting controlled imperfections, stylistic nuances, and natural "chaos," Entropy transforms robotic content into writing that feels authentically human.

## Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
